## ШАГ 1: Создание портативного контура VS Code
Изолированное действие:

   1. Зайди в свой рабочий каталог D:\Portable_Data_Suite\.
   2. Создай там новую пустую папку с именем vscode .
   3. Распакуй всё содержимое скачанного ZIP-архива VSCode-win32-x64-...zip внутрь этой созданной папки D:\Portable_Data_Suite\vscode\. (В корне папки должен появиться исполняемый файл Code.exe).
   4. Критическая точка контроля (Активация Portable-режима): Зайди внутрь папки D:\Portable_Data_Suite\vscode\. Создай там вручную новую пустую папку с именем data (строго маленькими буквами).

Физика процесса: Наличие пустой папки data в одном корне с файлом Code.exe является для VS Code системным триггером. Обнаружив её при старте, редактор мгновенно отключает работу с глобальными каталогами Windows (%USERPROFILE%/.vscode/extensions) и начинает складывать все устанавливаемые плагины, кэш и настройки внутрь этой папки. 

Как только папка data создана в одном корне с Code.exe. Перенос настроек и плагинов в изолированный режим является успешным.
Теперь нам нужно запустить редактор и установить официальное расширение Python, чтобы VS Code научился понимать синтаксис кода и автоматически связался с окружением Anaconda3.

## ШАГ 2: Первичный запуск и установка расширения Python
Изолированное действие:

   1. Зайди в папку D:\Portable_Data_Suite\vscode\ и запусти файл Code.exe.
   2. На левой вертикальной панели инструментов найди иконку Extensions (четыре квадрата, где один отлетает в сторону) и кликни по ней.
   3. В верхней строке поиска вбей руками: Python.
   4. Найди в результатах официальный плагин от Microsoft (рядом с названием будет стоять синяя галочка верификации) и нажми кнопку Install.
   5. Дождись, пока кнопка сменится на шестеренку (установка завершена).


## ШАГ 3: Принудительный досыл расширения Jupyter
Действие:

   1. На левой вертикальной панели VS Code  нажми на иконку Extensions (четыре квадрата).
   2. В строке поиска вбей руками: Jupyter.
   3. Найди официальный плагин от Microsoft (с синей галочкой верификации) и нажми кнопку Install.
   4. Дождись окончания установки (кнопка сменится на значок шестерёнки).
   5. Обязательно перезагрузи редактор: Закрой VS Code крестиком и запусти его заново через файл D:\Portable_Data_Suite\vscode\Code.exe .


## ШАГ 4: Создание рабочего блокнота Jupyter и выбор ядра
Действие:

   1. В верхнем меню VS Code нажми File → New File...
   2. В появившемся выпадающем окне выбери тип файла: Jupyter Notebook (или просто сохрани новый пустой файл с расширением .ipynb, например D:\Portable_Data_Suite\test.ipynb). Перед тобой откроется пустой холст с первой пустой ячейкой кода.
   3. Посмотри в правый верхний угол открывшегося блокнота. Там будет кнопка Select Kernel (Выбрать ядро). Кликни на неё.
   4. В выпадающем меню выбери пункт Python Environments...
   5. Система просканирует диск D и должна вывести строку с твоей изолированной средой: base (D:\Portable_Data_Suite\Anaconda3). Кликни строго по ней.
   6. Проверь правый верхний угол блокнота. Там должна зафиксироваться надпись: Python 3.13... (base).
