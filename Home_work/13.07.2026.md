## Шаг 1. Математический импульс

* Код: 2 + 2
* Физика процесса: Проверка работоспособности двигателя. Команда передана из редактора в RAM, интерпретатор выполнил сложение и вернул стерильный результат 4 на верстак.

## Шаг 2. Создание именованного ярлыка

* Код: HoursSMR = 8
* Физика процесса: В оперативной памяти компьютера зарезервирована ячейка, внутрь нее положено число 8, а на саму ячейку наклеен ярлык с именем HoursSMR. Класс данных определен системой автоматически как int (Integer — целое число).

## Шаг 3. Вызов данных на верстак

* Код: print(HoursSMR)
* Физика процесса: Базовая команда print() обратилась в RAM по ярлыку HoursSMR, извлекла значение и вывела на экран число 8.

## Шаг 4. Намеренная атака на синтаксис (Регистрозависимость)

* Код: print(hourssmr)

## Шаг 5. Динамическая перезапись памяти
```python
Код: HoursSMR = 11
```
* Физика процесса: Система вытеснила старое число 8 из зарезервированной ячейки памяти и принудительно записала туда новое число 11 под тем же ярлыком.

## Шаг 6. Капкан межклассового конфликта

* Код: Total = HoursSMR + "продленка"
* Физика процесса: Скрипт упал с логом TypeError: unsupported operand type(s) for +: 'int' and 'str'. Математический движок выдал аварию: нельзя прибавлять буквы (класс str) к числу (класс int). Ярлык Total в памяти не зафиксировался.

## Шаг 7. Принудительное изменение класса (Склеивание миров)

* Код:
```python
Total = str(HoursSMR) + " продленка"
```
* Физика процесса: Реализован фундаментальный закон изменения типов данных. Функция str() принудительно расплавила число 11 в текст "11". Оператор + успешно склеил две текстовые строки, выведя на верстак результат "11 продленка".

## Шаг 8. Заражение опасной пустотой None

* Код: HoursNone = None; Total = HoursSMR + HoursNone
* Физика процесса: Скрипт упал с логом TypeError: unsupported operand type(s) for +: 'int' and 'NoneType'. Имитация пропущенного табеля рабочего. В переменную записана системная пустота None. Математически прибавить пустоту к числу невозможно. Конвейер остановлен аварийным сигналом.

## Шаг 9. Защита Мастера (Логический перехват пустоты)

* Код:
```python
HoursNone = None

if HoursNone is None:
    SafeHours = 0
else:
    SafeHours = HoursNone

Total = HoursSMR + SafeHours
print(Total)
```
* Базовая конструкция if-else в Python проверяет условие и направляет ход программы по одной из ветвей. После условия обязательно ставится двоеточие (:), а блоки кода выделяются отступами в 4 пробела

* Физика процесса: Написано логическое сито-предохранитель (базовый аналог оператора COALESCE в SQL). Код перехватил None до момента сложения и создал безопасную переменную SafeHours, принудительно записав в нее число 0. Скрипт успешно завершил вычисления и выдал на экран спасенный результат 11.
